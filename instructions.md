# How to play

### Movement:

| Key        |   | Action   |
|------------|---|----------| 
|            |   |          |
|     Up/E    |   | Move Up  |
|    Down/D      |   | Move Down|
|     Left/S    |   | Move left|
|     Right/F   |   | Move right|
| Left Mouse || Throw Knife
| Right Mouse || Shunpo



- Note: These controls are subject to change
---
### Known bugs: Right mouse attack does not connect with enemies or work at all.
