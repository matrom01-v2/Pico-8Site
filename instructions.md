# How to play

### Movement:

| Key        |   | Action   |
|------------|---|----------| 
|            |   |          |
|     Up     |   | Move Up  |
|    Down      |   | Move Down|
|     Left    |   | Move left|
|     Right   |   | Move right|
|   X | | Attack/Ability
|   C || Change ability

- Note: These controls are subject to change
---
### Known bugs: Right mouse attack does not connect with enemies
